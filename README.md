# PyForge Project Lab

A dependency-free gamified web app for revisiting intermediate Python by building one end-to-end project: ProjectPulse, a CLI issue tracker with persistence, imports, analytics, tests, typing, async enrichment, and packaging.

Open `index.html` in a browser to run the app. Progress, XP, badges, theme, and review marks are stored in browser `localStorage`.

## Build Phases

- Blueprint
- Core Engine
- Data Layer
- Interfaces
- Quality And Ship

The app includes project deliverables, build checklists, checkpoint challenges, a mixed review arena, concept cards, a progress map, and badge unlocks.
